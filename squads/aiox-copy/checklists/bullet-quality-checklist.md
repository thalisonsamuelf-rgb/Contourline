# bullet-quality-checklist

Checklist completo de qualidade para bullets/fascinations - baseado na metodologia Parris Lampropoulos.

---

## SECTION 1: THE FOREHEAD-SLAPPER TEST

### Reação Imediata

Para CADA bullet, pergunte:

- [ ] Senti um impulso físico de saber mais?
- [ ] Pagaria R$20 só para descobrir esta resposta?
- [ ] Ficaria desapontado se a resposta fosse óbvia?
- [ ] Parece um segredo que eu não deveria saber?

**Scoring Imediato:**
- 4 SIMs = ✅ Bullet forte
- 3 SIMs = 🔄 Precisa polish
- 2 ou menos = ❌ Reescreva ou elimine

---

## SECTION 2: SPECIFICITY CHECK

### Números e Detalhes

- [ ] Contém número específico (não arredondado)?
  - ✅ "47%" não ❌ "quase metade"
  - ✅ "R$847" não ❌ "centenas de reais"
  - ✅ "em 17 dias" não ❌ "em poucas semanas"

- [ ] Inclui nomes específicos?
  - ✅ "Dr. Mark Hyman, Harvard"
  - ✅ "Estudo Johns Hopkins 2023"
  - ❌ "especialistas dizem"

- [ ] Referencia páginas/seções quando aplicável?
  - ✅ "...página 47"
  - ✅ "...capítulo 3"

- [ ] Detalhes visuais permitem imaginar?
  - ✅ "O exercício de 3 minutos que você faz na cama"
  - ❌ "Um exercício simples"

### Red Flags de Vagueza

Elimine bullets com estas palavras vagas:
- [ ] "muitos" / "vários" / "alguns"
- [ ] "frequentemente" / "geralmente"
- [ ] "pode ajudar" / "pode funcionar"
- [ ] "significativo" / "importante"
- [ ] "especialistas" (sem nome)

---

## SECTION 3: CURIOSITY GAP CHECK

### Criação de Lacuna

- [ ] Há uma lacuna clara entre o que sabem e o que querem saber?
- [ ] O leitor tem um palpite—mas não tem certeza?
- [ ] Se sentiria "incompleto" não sabendo?
- [ ] O bullet levanta mais perguntas do que responde?

### Técnicas de Curiosidade Presentes

- [ ] **Misdirection aplicada?**
  - Hint na direção errada
  - Categoria inesperada
  - Fonte surpresa

- [ ] **"Call it something else" usado?**
  - Nome intrigante em vez de termo comum
  - Autoridade emprestada no nome

- [ ] **Paradoxo criado?**
  - Contradição aparente
  - "Por que X faz o oposto do esperado"

### Formas de Curiosidade

Cada bullet deve usar pelo menos uma:

- [ ] Segredo revelado ("O que X não conta")
- [ ] Conhecimento proibido ("O que você nunca deve...")
- [ ] Descoberta surpreendente ("O link inesperado entre...")
- [ ] Erro comum ("Você está cometendo este erro?")
- [ ] Insider knowledge ("O que experts fazem para si mesmos")

---

## SECTION 4: BENEFIT CHECK

### Clareza de Benefício

- [ ] O "What's In It For Me" está claro?
- [ ] O benefício é significativo para o avatar?
- [ ] A transformação prometida é desejável?
- [ ] O benefício justificaria pagar pelo produto?

### Tipos de Benefício

Verifique se o benefício é:

- [ ] **Funcional** - Resolve problema prático
- [ ] **Emocional** - Faz sentir algo positivo
- [ ] **Social** - Melhora status/relacionamentos
- [ ] **Financeiro** - Economiza/ganha dinheiro
- [ ] **Tempo** - Poupa tempo/esforço
- [ ] **Saúde** - Melhora bem-estar físico
- [ ] **Segurança** - Reduz risco/protege

### Quantificação

- [ ] Benefício é quantificado quando possível?
  - ✅ "Corta risco de infarto em 47%"
  - ❌ "Melhora saúde do coração"

- [ ] Timeframe incluído quando relevante?
  - ✅ "...em apenas 21 dias"
  - ❌ "...rapidamente"

---

## SECTION 5: PROOF CHECK

### Credibilidade Embutida

- [ ] O bullet carrega sua própria credibilidade?
- [ ] Há autoridade emprestada?
- [ ] Prova social presente?
- [ ] Fonte científica citada?

### Tipos de Prova no Bullet

- [ ] **Estudo/Pesquisa**
  - "Segundo estudo de Harvard..."
  - "Pesquisa com 10.000 pessoas revelou..."

- [ ] **Expert/Autoridade**
  - "O que cardiologistas fazem..."
  - "Recomendado por Dr. X de Y..."

- [ ] **Números de Prova Social**
  - "Usado por 47.000 pessoas..."
  - "94% de taxa de sucesso..."

- [ ] **Autoridade Institucional**
  - "Aprovado pela FDA..."
  - "Reconhecido por Johns Hopkins..."

- [ ] **Testemunho Implícito**
  - "O homem de 87 anos que..."
  - "A mãe de 3 que..."

---

## SECTION 6: CRAFT & RHYTHM CHECK

### Estrutura

- [ ] Termina em palavra de poder ou consoante forte?
  - ✅ "...que DERRETE gordura"
  - ❌ "...que pode ajudar com a perda de peso"

- [ ] Voz ativa usada?
  - ✅ "Este alimento DERRUBA colesterol"
  - ❌ "Colesterol pode ser reduzido por..."

- [ ] Comprimento apropriado?
  - Muito curto = sem contexto suficiente
  - Muito longo = perde impacto

### Economia de Palavras

- [ ] Adjetivos desnecessários removidos?
- [ ] Verbos fracos substituídos por fortes?
  - get → pegue, capture
  - make → dispare, gatilho
  - help → desbloqueie, ative

- [ ] Palavras de hedge eliminadas?
  - pode, talvez, poderia, provavelmente → REMOVIDOS

### Teste de Leitura em Voz Alta

- [ ] Soa natural quando lido em voz alta?
- [ ] Tem ritmo/cadência agradável?
- [ ] Flow sem tropeços?

---

## SECTION 7: TYPE DIVERSITY CHECK

### Variedade de Tipos

No conjunto completo de bullets, verifique diversidade:

**Bullets How-To**
- [ ] How-To Básico presente?
- [ ] How-To Secreto presente?

**Bullets de Segredo**
- [ ] Segredo Revelado presente?
- [ ] Verdade Escondida presente?

**Bullets de Aviso**
- [ ] Warning presente?
- [ ] Mistake/Erro presente?

**Bullets de "Por Quê"**
- [ ] Why Surpreendente presente?
- [ ] Why Funciona presente?

**Bullets de "O Que"**
- [ ] What Never presente?
- [ ] What Always presente?

**Bullets de Número**
- [ ] Lista de Números presente?
- [ ] Número Específico presente?

**Bullets de Pergunta**
- [ ] "Are You Making" presente?
- [ ] "Do You Know" presente?

**Bullets de História**
- [ ] Story Tease presente?
- [ ] Doctor/Expert Secret presente?

**Bullets Avançados**
- [ ] Paradoxo presente?
- [ ] Surprising Link presente?
- [ ] Study Says presente?
- [ ] Proof with Number presente?
- [ ] Renamed/Reframed presente?

**Target:** Mínimo 10 tipos diferentes no conjunto final

---

## SECTION 8: ORGANIZATION CHECK

### Categorização por Uso

- [ ] **COVER BULLETS (10-15)**
  - Os mais fortes identificados?
  - Funcionam sem contexto?
  - Máxima curiosidade?

- [ ] **TOC BULLETS (15-20)**
  - Referências de página incluídas?
  - Variedade de tipos?
  - Cobrem diferentes aspectos?

- [ ] **BODY BULLETS (50+)**
  - Organizados por seção/capítulo?
  - Suportam argumento principal?
  - Progressão lógica?

- [ ] **OFFER BULLETS (20-30)**
  - Focados em benefícios específicos?
  - Eliminam objeções?
  - Criam urgência?

### Priorização

- [ ] Top 15 bullets claramente identificados?
- [ ] Bullets similares comparados—melhor mantido?
- [ ] Reserve bullets documentados?

---

## SECTION 9: VOLUME CHECK

### Processo 700-to-100

- [ ] Pelo menos 300 bullets foram gerados inicialmente?
  - Ideal: 700+ (método Lampropoulos)
  - Mínimo aceitável: 300

- [ ] Múltiplas variações por fato/benefício?
  - Cada fato tentado em 3+ tipos diferentes?

- [ ] Seleção final de 100+?
  - Bullets suficientes para todas as seções?

### Documentação

- [ ] Fatos originais documentados com fonte?
- [ ] Páginas de referência anotadas?
- [ ] Processo de seleção registrado?

---

## SECTION 10: FINAL QUALITY SCORE

### Scoring por Bullet

Para cada bullet, some pontos:

| Critério | Pontos |
|----------|--------|
| Passa no Forehead Test | +2 |
| Especificidade presente | +2 |
| Curiosity Gap criado | +2 |
| Benefício claro | +1 |
| Prova embutida | +1 |
| Morpheus excelente | +1 |
| Originalidade | +1 |

**Total por bullet: /10**

### Thresholds de Qualidade

- **9-10 pontos:** Bullet A+ (Top 15 - use no cover)
- **7-8 pontos:** Bullet A (Top 50 - use em destaque)
- **5-6 pontos:** Bullet B (Aceitável para body)
- **3-4 pontos:** Bullet C (Reescreva ou elimine)
- **0-2 pontos:** Elimine imediatamente

### Score Geral do Conjunto

```markdown
## QUALITY SCORE SUMMARY

Total de bullets finais: ___

Distribuição:
- A+ (9-10): ___ bullets (___ %)
- A (7-8):   ___ bullets (___ %)
- B (5-6):   ___ bullets (___ %)
- C (3-4):   ___ bullets (___ %)

Average Score: ___ /10

Target: Average 7.0+ com pelo menos 15 bullets A+
```

## QUICK REFERENCE

### O Teste do Tapa na Testa (4 Perguntas)

1. Senti impulso físico de saber mais?
2. Pagaria R$20 só por esta resposta?
3. Ficaria desapontado se fosse óbvio?
4. Parece um segredo que não deveria saber?

### A Fórmula Mestre

```
ESPECIFICIDADE + CURIOSIDADE + BENEFÍCIO + PROVA = Bullet Irresistível
```

### Red Flags Instantâneos (Elimine Se Presente)

- ❌ Começa com "Aprenda sobre..."
- ❌ Contém "pode ajudar"
- ❌ Usa "especialistas dizem" (sem nome)
- ❌ Números arredondados (50%, 100, etc.)
- ❌ Sem benefício claro
- ❌ Voz passiva
- ❌ Mais de 25 palavras sem punch

### Palavras de Poder para Finais

VERBOS: derrete, dispara, derruba, destrói, desbloqueia, ativa, revela, expõe

SUBSTANTIVOS: segredo, truque, hack, método, fórmula, protocolo, sistema

ADJETIVOS: proibido, escondido, perdido, secreto, surpreendente, chocante
