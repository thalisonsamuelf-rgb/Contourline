# Call Script Quality Checklist

## Overview

Quality gate for sales call scripts generated by the Copy Squad. Validates rapport building, qualification, objection handling, and closing mechanics for phone or video sales calls.

---

## Critical Checks (must pass ALL)

### Rapport Building
- [ ] Opening 60 seconds establish warmth without being fake?
- [ ] Script includes discovery questions about the prospect's situation?
- [ ] Caller mirrors language and energy (notes for vocal direction)?
- [ ] Transition from rapport to qualification is natural (not abrupt)?
- [ ] Prospect feels heard before any pitching begins?

### Qualification
- [ ] Clear criteria to identify qualified vs unqualified prospects?
- [ ] Questions uncover the prospect's pain, desire, and timeline?
- [ ] Prospect self-identifies their problem (not told by the caller)?
- [ ] Budget/authority/need/timeline (BANT) are addressed?
- [ ] Disqualification path is scripted (graceful exit for non-fits)?

### Objection Handling
- [ ] Top 5 common objections have scripted responses?
- [ ] Objection responses use "feel, felt, found" or equivalent framework?
- [ ] Each response ends with a question (not a monologue)?
- [ ] "I need to think about it" has a specific handling protocol?
- [ ] "It's too expensive" has value reframing, not discounting?

### Trial Closes
- [ ] Micro-commitments are built throughout the call?
- [ ] At least 3 trial close opportunities before the final ask?
- [ ] Trial closes use assumptive language ("when you start..." not "if...")?
- [ ] Prospect agreement is confirmed at each stage before advancing?

### Urgency Creation
- [ ] A legitimate reason to decide today is presented?
- [ ] Consequence of delay is personalized to the prospect's situation?
- [ ] Urgency relates to the prospect's own timeline and goals?
- [ ] No pressure tactics that damage trust?

---

## Red Flags (auto-fail if present)

- Script reads as a monologue with no prospect interaction points
- No objection handling section for common pushbacks
- Opens with a pitch instead of discovery
- Only one closing attempt (all-or-nothing)
- No disqualification path (tries to sell everyone)
- Pressure tactics that would damage the brand
- Script is so rigid it cannot adapt to real conversation flow

---

## Call Script Technical Checks

- [ ] Talk-to-listen ratio targets 30/70 (caller talks 30%, listens 70%)?
- [ ] Script includes vocal direction notes (pause here, lower voice, speed up)?
- [ ] Branching paths are clearly marked for different prospect responses?
- [ ] Total estimated call duration is noted (typically 30-60 min for high-ticket)?
- [ ] Follow-up sequence is defined for non-closers?
- [ ] CRM notes/tags are specified for each call outcome?

---

## Minimum Score

Total >= 35/50 for approval (70% threshold)
