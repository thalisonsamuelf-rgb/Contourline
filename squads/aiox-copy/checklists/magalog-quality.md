# Magalog Quality Checklist

## Overview

Quality gate for magalogs (magazine-style sales letters) generated by the Copy Squad. Validates layout flow, editorial authority, proof density, and conversion elements.

---

## Critical Checks (must pass ALL)

### Layout Flow
- [ ] Cover headline stops the reader cold?
- [ ] Table of contents or teaser panel draws reader inside?
- [ ] Editorial content and sales content are seamlessly blended?
- [ ] Eye path follows a natural reading pattern (Z-pattern or F-pattern)?
- [ ] Page turns create curiosity to continue?

### Sidebar Usage
- [ ] Sidebars contain proof elements (testimonials, stats, callouts)?
- [ ] Sidebars break up dense body copy at regular intervals?
- [ ] Pull quotes highlight the strongest claims?
- [ ] Sidebars do not compete with the main narrative?

### Headline Hierarchy
- [ ] Cover headline is the single strongest promise?
- [ ] Section headlines could sell on their own (not decorative)?
- [ ] Sub-headlines support scanning and skimming behavior?
- [ ] Headline sizes create clear visual hierarchy?
- [ ] Internal headlines re-hook attention at each spread?

### Proof Density
- [ ] At least one proof element per spread (testimonial, stat, chart, photo)?
- [ ] Proof is specific (names, dates, numbers) not generic?
- [ ] Mix of proof types (social proof, expert proof, data proof)?
- [ ] Proof placement supports the adjacent copy claim?

### Order Form Clarity
- [ ] Order form could work as a standalone sales piece?
- [ ] Every offer element is restated on the order form?
- [ ] Guarantee is prominently displayed on the order form?
- [ ] Response mechanism is frictionless and clearly explained?
- [ ] Urgency element is visible on the order form?

---

## Red Flags (auto-fail if present)

- Reads like a sales letter with pictures (no editorial disguise)
- Order form is an afterthought with missing details
- No proof elements for 2+ consecutive pages
- Headlines are generic labels ("Chapter 1", "Introduction")
- Sidebars are empty filler or decorative only
- No visual break for 3+ consecutive text-heavy pages
- Cover looks like an ad instead of a publication

---

## Agent-Specific Validation

- If generated by **Gary Bencivenga agent**: validate proof-first approach, credibility layering
- If generated by **David Ogilvy agent**: validate editorial tone, factual density, "long copy that reads short"
- If generated by **John Carlton agent**: validate hard-hitting subheads and salesmanship-in-print craft

---

## Magalog-Specific Technical Checks

- [ ] Page count is appropriate (typically 16-32 pages)?
- [ ] Paper stock and format notes are included for print production?
- [ ] Images and charts are captioned (captions are read 2x more than body)?
- [ ] Response device (URL, QR code, phone number) appears on multiple pages?

---

## Minimum Score

Total >= 42/60 for approval (70% threshold)
