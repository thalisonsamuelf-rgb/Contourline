# Decision Matrix Quality Checklist

## Overview

Quality gate for decision matrices (comparison copy) generated by the Copy Squad. Validates fairness of comparison, feature selection, scoring transparency, and the strategic positioning that creates an "obvious winner" effect.

---

## Critical Checks (must pass ALL)

### Comparison Fairness
- [ ] Competitors are represented accurately (no strawmen)?
- [ ] All options compared actually exist and are relevant alternatives?
- [ ] Competitor strengths are acknowledged (builds credibility)?
- [ ] Data points are verifiable and current?
- [ ] Comparison is apples-to-apples on each dimension?

### Feature Selection
- [ ] Features chosen are genuinely important to the target buyer?
- [ ] Feature list is not cherry-picked to only favor the promoted option?
- [ ] Features cover different categories (price, quality, support, speed)?
- [ ] No obscure features included solely because the promoted option wins?
- [ ] Missing features that competitors are known for are addressed?

### Scoring Transparency
- [ ] Scoring methodology is explained (how ratings are determined)?
- [ ] Scores are justified with evidence or reasoning?
- [ ] Rating scale is consistent across all options?
- [ ] Sources for claims are cited or available?
- [ ] Reader understands why each score was given?

### "Obvious Winner" Effect
- [ ] Promoted option wins clearly but credibly?
- [ ] Win is not suspiciously lopsided (10/10 on everything)?
- [ ] Promoted option loses on 1-2 dimensions (builds trust)?
- [ ] Overall advantage is demonstrated through the buyer's priorities?
- [ ] Conclusion is drawn from the data, not asserted?

---

## Red Flags (auto-fail if present)

- Promoted option scores perfectly on every dimension
- Competitors are misrepresented or outdated
- Features are clearly cherry-picked to rig the outcome
- No acknowledgment of any area where competitors are strong
- Scoring methodology is hidden or inconsistent
- Claims are unverifiable
- Matrix looks like an ad disguised as editorial comparison

## Matrix Format Checks

### Table Structure
- [ ] Headers are clear and descriptive?
- [ ] Visual indicators (checkmarks, X marks, scores) are consistent?
- [ ] Promoted option is visually highlighted but not obnoxiously?
- [ ] Table is readable on mobile (not too many columns)?
- [ ] Summary row or "bottom line" is included?

### Contextual Elements
- [ ] Introduction explains who the comparison is for?
- [ ] Buyer persona is clear (different buyers may have different "winners")?
- [ ] "Best for" qualifier is used for each option?
- [ ] Conclusion paragraph ties the matrix to a CTA?
- [ ] Disclaimer about methodology or last-updated date is present?

## The Trust Test

| Question | Yes/No |
|----------|--------|
| Would a journalist publish this comparison without edits? | |
| Would a competitor accept the representation as fair? | |
| Would a skeptical buyer find it credible? | |
| Is there at least one area where the promoted option is NOT the best? | |

3/4 yes required.
