# Cold vs Warm Webinar Decision Checklist

## Purpose

Decidir qual tipo de webinar usar (cold ou warm) e validar que a execução está alinhada com as melhores práticas de cada tipo.

---

## Parte 1: Diagnóstico - Qual Tipo de Webinar?

### Identifique Sua Audiência

**Warm Audience** = qualquer um destes:
- [ ] Website visitors (180 dias)
- [ ] Email list subscribers
- [ ] Customer list
- [ ] Social interaction (365 dias)
- [ ] Video viewers (25%+ threshold)
- [ ] Profile visitors
- [ ] Post engagers
- [ ] Ad clickers/engagers
- [ ] Seguidores da página
- [ ] Pessoas que assistiram stories

**Cold Audience** = NENHUM dos acima:
- [ ] Nunca interagiu com você
- [ ] Não está em nenhuma lista
- [ ] Targeting por interesses/lookalike
- [ ] Completamente desconhecido

### Decisão

```
Se TODAS as warm audiences estão EXCLUÍDAS no targeting → COLD WEBINAR
Se audiência inclui qualquer warm audience → WARM WEBINAR
```

**Seu webinar é:** [ ] COLD [ ] WARM

---

## Parte 2: Checklist - COLD Webinar

### Frequência & Promoção

- [ ] Frequência: **Semanal** ou até 2x/semana?
- [ ] Período de promoção: **Máximo 7 dias**?
- [ ] Warm audiences **EXCLUÍDAS** no targeting?
- [ ] Ads **não mencionam "webinar"** (usar "training/workshop")?
- [ ] Ads **não têm datas** (evergreen)?

### Intro (30 segundos MÁXIMO)

- [ ] **ZERO** "de onde vocês são"?
- [ ] **ZERO** "manda 1 no chat"?
- [ ] **ZERO** small talk?
- [ ] **Imediatamente** diz o que vão aprender?
- [ ] Menciona **exemplos reais**?
- [ ] Transição rápida para conteúdo?

### Estrutura de 3 Pitches

**Pitch #1 (após primeiro terço):**
- [ ] Pediu **permissão** para pitchar?
- [ ] Duração: **2-5 minutos**?
- [ ] Apresentou: produto, preço, componentes?
- [ ] Voltou **imediatamente** para conteúdo?

**Pitch #2 (após segundo terço):**
- [ ] Duração: **2-5 minutos**?
- [ ] Foco em **framing** (comprar velocidade/suporte)?
- [ ] **NÃO repetiu** o que vem no produto?
- [ ] Voltou para conteúdo?

**Pitch #3 (terço final):**
- [ ] Duração: **10-15 minutos**?
- [ ] Inclui **objection handling** completo?
- [ ] **Full pitch** com stack de valor?
- [ ] CTA claro?
- [ ] Q&A extenso após?

### Conteúdo

**First Third (Surface Level):**
- [ ] Visão **high-level** do framework?
- [ ] O QUÊ e POR QUÊ (não HOW)?
- [ ] **1-2 exemplos rápidos**?

**Second Third (Examples):**
- [ ] **COMO funciona** com exemplos?
- [ ] Case studies com **números específicos**?
- [ ] Demonstrações se aplicável?

**Third Third (Deep/Action):**
- [ ] Conteúdo mais **profundo**?
- [ ] **Exercício ou demonstração** que faz começar?
- [ ] Revela complexidades que precisam de ajuda?

### Mindset Cold

- [ ] Lembrei que cold **não dá a mínima** para mim?
- [ ] Lembrei que são **cruéis** e vão embora rápido?
- [ ] Lembrei de manter **value density extrema**?
- [ ] **Zero character content** (sem piadas internas, etc.)?

### Show Rate Cold

- [ ] Show rate esperado: **15-45%**?
- [ ] Hammer Them Strategy ativada?
- [ ] Email sequence pré-webinar configurada?
- [ ] Confirmation page com video?

---

## Parte 3: Checklist - WARM Webinar

### Frequência & Promoção

- [ ] Frequência: **Mensal** (máx 2x/mês)?
- [ ] Período de promoção: **14 dias**?
- [ ] Audiências warm corretamente targetadas?
- [ ] Não vai saturar a audiência?

### Intro (Pode ter personalidade)

- [ ] Abertura com **personalidade** (mas não longa)?
- [ ] Pode ter **referências** a conteúdo anterior?
- [ ] Máximo **5 minutos** de abertura?
- [ ] Ainda vai para **valor** rapidamente?

### Estrutura (Mais flexível)

- [ ] Pode ter **pitch mais no final**?
- [ ] Pode ter **mais storytelling**?
- [ ] Ainda tem **structure clara**?
- [ ] Pitch bem definido mesmo que único?

### Conteúdo

- [ ] **Character content** apropriado?
- [ ] Pode usar **humor** e estilo pessoal?
- [ ] Referências a **conteúdo/eventos anteriores**?
- [ ] Ainda **value-dense**?

### Duração

**Affluent ($150k+/ano):**
- [ ] Máximo **60-90 minutos**?
- [ ] Estilo **estratégico/high-level**?
- [ ] Respeita tempo deles?

**General Public:**
- [ ] Pode ser **90-180 minutos**?
- [ ] Estilo mais **tático/detalhado**?
- [ ] Mais hand-holding?

### Mindset Warm

- [ ] Lembrei que warm **≠ amigos**?
- [ ] Ainda precisa **provar valor**?
- [ ] Personalidade sim, **enrolação não**?
- [ ] Não vou saturar com frequência alta?

### Show Rate Warm

- [ ] Show rate esperado: **40-60%**?
- [ ] Email sequence configurada?
- [ ] Retargeting ativo?

---

## Parte 4: Comparativo Rápido

| Aspecto | Cold | Warm |
|---------|------|------|
| Frequência | Semanal/2x semana | Mensal |
| Promoção | 7 dias máx | 14 dias |
| Intro | ZERO small talk | Pode ter personalidade |
| Pitches | 3 durante o webinar | Pode ser 1 no final |
| Character | Evitar | Encorajado |
| Show rate | 15-45% | 40-60% |
| Framing | Evitar | Pode usar |
| Duração | 60 min (máx 90) | Flexível por audiência |

---

## Parte 5: Erros Comuns

### Cold Webinar - NÃO FAZER:

- [ ] ❌ Tratar cold como se fossem warm
- [ ] ❌ Começar com small talk
- [ ] ❌ Fazer só 1 pitch no final
- [ ] ❌ Usar framing content (secrets, beliefs)
- [ ] ❌ Webinar mensal para cold (pode ser semanal)
- [ ] ❌ Não excluir warm audiences no targeting
- [ ] ❌ Character content (piadas, etc.)

### Warm Webinar - NÃO FAZER:

- [ ] ❌ Frequência muito alta (fadiga)
- [ ] ❌ Assumir que são "amigos"
- [ ] ❌ Abandonar value density
- [ ] ❌ Webinar muito curto (perder oportunidade)
- [ ] ❌ Ignorar que ainda precisam ser convencidos
