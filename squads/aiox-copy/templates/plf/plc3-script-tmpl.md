# PLC3 Script Template - The Ownership Experience

> **Framework**: Product Launch Formula (Jeff Walker)
> **PLC3 Focus**: HOW - Make them feel they already own it
> **Duration**: 25-45 minutes of content
> **Triggers to Activate**: Anticipation, Community, Events

---

## Video/Content Structure

### HOOK + RECAP (0:00 - 3:00)
**Goal**: Bridge both previous PLCs, set up ownership

```
Nos ultimos dois videos, voce aprendeu:

Video 1: {{RECAP_PLC1}} - A oportunidade
Video 2: {{RECAP_PLC2}} - A prova de que funciona

Hoje e diferente.

Hoje nao vou apenas te ensinar algo...
Vou fazer voce SENTIR como sera quando {{RESULTADO}}.

E vou te preparar para o que vem amanha.
```

**Alternative Hooks:**
```
Hook A (Vision):
"Imagina acordar daqui {{TEMPO}} sabendo que {{RESULTADO}}..."

Hook B (Direct):
"Este e o ultimo video antes de eu abrir as inscricoes.
Mas antes disso, preciso te mostrar algo importante."

Hook C (Question):
"Voce esta pronto para dar o proximo passo?"
```

### OWNERSHIP VISION (3:00 - 10:00)
**Goal**: Make them visualize the transformed future

```
Vou te pedir para fechar os olhos por um momento.
(Ou apenas imaginar comigo)

Imagina que e {{TEMPO_FUTURO}} - daqui a {{X MESES}}.

Voce acorda e...
- {{ELEMENTO_DA_NOVA_REALIDADE_1}}
- {{ELEMENTO_DA_NOVA_REALIDADE_2}}
- {{ELEMENTO_DA_NOVA_REALIDADE_3}}

Em vez de {{FRUSTRACAO_ANTIGA}}...
Voce {{NOVA_REALIDADE}}.

Voce olha para tras e lembra de hoje.
De quando tomou a decisao de {{ACAO}}.
Foi o ponto de virada.

Isso nao e fantasia.
E exatamente o que {{NOME_DE_CASE}} experimentou.

"{{QUOTE_SOBRE_TRANSFORMACAO}}"
```

**Ownership Elements:**
- [ ] Sensory details (what they see, feel, hear)
- [ ] Daily life transformation
- [ ] Emotional state change
- [ ] Relationships impacted
- [ ] Validation from case study

### FINAL TEACHING (15:00 - 28:00)


### BRIDGE TO OFFER (28:00 - 35:00)
**Goal**: Transition from value to opportunity

```

- {{VALOR_PLC3}}

Mas se voce quiser...
```

### PRODUCT PREVIEW (35:00 - 40:00)
**Goal**: Show what's inside without full pitch

```

Modulo 1: {{NOME}}
- {{O_QUE_VAO_APRENDER}}

Modulo 2: {{NOME}}
- {{O_QUE_VAO_APRENDER}}
```

### URGENCY + SCARCITY SETUP (40:00 - 43:00)
**Goal**: Prepare for cart open

```
Preciso ser transparente com voce:

O {{PRODUTO}} vai estar disponivel por tempo limitado.

{{RAZAO_DA_ESCASSEZ}}

Depois de {{DATA}}, a inscricao fecha.
E nao tenho previsao de quando reabre.

Entao se voce esta considerando:
- Esteja atento amanha as {{HORA}}
- O email vai ter o assunto "{{SUBJECT_DO_OPEN_CART}}"
- As vagas sao limitadas a {{X}} pessoas

Nao deixe para ultima hora.
```

---

### CTA + CLOSE (43:00 - 45:00)
**Goal**: Final engagement before cart

```
Antes de terminar:

Eu quero saber de voce.

Depois desses 3 videos:
- O que mais te impactou?
- O que voce vai implementar primeiro?
- Voce esta pronto para o proximo passo?

Comenta abaixo.

E se voce conhece alguem que precisa ouvir isso antes de amanha,
compartilha agora. E a ultima chance antes das inscricoes.

Nos vemos amanha as {{HORA}}.

Ate la,
{{ASSINATURA}}
```

## Ownership Techniques

**Visualization Tactics:**
- [ ] Use present tense ("You wake up...")
- [ ] Include sensory details
- [ ] Reference specific time frame
- [ ] Tie to their current pain
- [ ] Validate with real example

**Day in the Life Elements:**
- Morning routine
- Work/professional context
- Personal relationships
- Evening/reflection
- Emotional state
