# Create Campaign — New Campaign Setup & Launch

## Process

1. **Gather context** -- Collect from user:
   - Platform (Meta, Google, LinkedIn, TikTok, etc.)
   - Objective (Sales, Leads, Traffic, App Installs)
   - Daily/monthly budget
   - Target CPA or ROAS
   - Product/service description
   - Landing page URL
2. **Select funnel** -- Map objective to funnel stage:
   - TOF: Broad/LAL prospecting
   - MOF: Engagement retargeting
   - BOF: Website visitor retargeting, cart abandoners
3. **Structure campaign** -- Design architecture:
   - CBO vs ABO selection based on budget
   - Ad set segmentation (audience, placement)
   - Ad variations (3-6 per ad set minimum)
4. **Verify pre-launch checklist**:
   - Tracking pixel verified (delegate to pixel-specialist if needed)
   - Creative specs match platform requirements
   - Special Ad Categories checked (housing, employment, credit)
   - Attribution window configured
5. **Launch** -- Document campaign structure in output file

## Output

Write campaign structure document with IDs, targeting, budget allocation, and launch checklist status.
