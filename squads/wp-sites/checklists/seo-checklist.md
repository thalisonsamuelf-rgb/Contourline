# SEO Checklist

**Checklist ID:** `seo-checklist`
**Phase:** Development
**Agent:** wp-developer

## On-Page SEO
- [ ] Title tags unique and within 60 characters per page
- [ ] Meta descriptions unique and within 160 characters per page
- [ ] One H1 per page (only one)
- [ ] H2-H3 hierarchy is logical (no skipped levels)
- [ ] Focus keyword present in title, H1, and first paragraph
- [ ] Image alt text is descriptive (not keyword-stuffed)
- [ ] Internal links between related pages
- [ ] Outbound links to authoritative sources (where relevant)

## Technical SEO
- [ ] XML sitemap generated and submitted to Search Console
- [ ] Robots.txt configured (allows crawling of important pages)
- [ ] Canonical URLs set on all pages
- [ ] No duplicate content issues
- [ ] Clean URL structure (no ?p=123)
- [ ] SSL active (HTTPS)
- [ ] No mixed content
- [ ] 301 redirects configured (if migrating from old site)
- [ ] Page speed optimized (LCP < 2.5s)
- [ ] Mobile-friendly (passes Google Mobile-Friendly Test)

## Schema Markup
- [ ] Organization schema
- [ ] LocalBusiness schema (if local business)
- [ ] Breadcrumb schema
- [ ] FAQ schema (if FAQ section exists)

## Search Console
- [ ] Property verified
- [ ] Sitemap submitted
- [ ] No crawl errors
- [ ] No security issues
