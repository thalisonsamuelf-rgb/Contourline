# Design Review Checklist

**Checklist ID:** `design-checklist`
**Phase:** Design
**Agent:** wp-designer
**Gate:** Client approval required before proceeding to Build

## Asimov Academy Style Compliance
- [ ] Typography scale defined (H1 through caption, desktop + mobile)
- [ ] Font is geometric sans-serif (Inter or similar)
- [ ] Neutral color palette: white background, dark text
- [ ] Single accent color (client brand color)
- [ ] No more than 7 color tokens total
- [ ] Generous white space between sections (80-120px desktop)
- [ ] Single CTA per section with accent color

## Typography
- [ ] H1: 48-64px desktop / 32-40px mobile / 700 weight
- [ ] H2: 36-48px desktop / 28-32px mobile / 700 weight
- [ ] H3: 24-32px desktop / 20-24px mobile / 600 weight
- [ ] Body: 16-18px desktop / 16px mobile / 400 weight
- [ ] Line height: 1.2 headings, 1.6 body

## Responsive Design
- [ ] Mobile breakpoint defined (375px)
- [ ] Tablet breakpoint defined (768px)
- [ ] Desktop breakpoint defined (1200px)
- [ ] Every section has behavior documented at all 3 breakpoints
- [ ] Mobile stack order explicitly defined
- [ ] CTA buttons are full-width on mobile

## Components
- [ ] CTA button spec complete (bg, text, padding, radius, hover)
- [ ] Card spec complete (bg, padding, radius, shadow)
- [ ] Testimonial format defined

## Layout Specs
- [ ] Layout spec exists for every page section
- [ ] Max-width container defined (1200px)
- [ ] Content padding defined (desktop + mobile)
- [ ] Grid patterns documented (benefits, services, etc.)

## Final Validation
- [ ] Visual direction document complete
- [ ] Layout specs complete for all pages
- [ ] Specs are implementable in Elementor/Gutenberg
- [ ] Client has reviewed the design direction
- [ ] Client has approved (gate)
