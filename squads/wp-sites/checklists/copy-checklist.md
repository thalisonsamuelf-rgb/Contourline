# Copy Quality Checklist

**Checklist ID:** `copy-checklist`
**Phase:** Copywriting
**Agent:** wp-copywriter
**Gate:** Client approval required before proceeding to Design

## Asimov Academy Style Compliance
- [ ] All H1 headlines start with an action verb
- [ ] All H1 headlines are 8 words or fewer
- [ ] Subheadlines provide context in 12-15 words
- [ ] All CTAs are specific (no "Learn More", "Click Here", "Submit")
- [ ] CTAs start with a verb
- [ ] One CTA per section
- [ ] Short paragraphs (2-3 sentences max)
- [ ] Benefits framed as problem-solution (not feature lists)
- [ ] Second person voice (you/your)

## Hero Copy
- [ ] Every page has hero copy (H1 + subheadline + CTA)
- [ ] 2-3 alternative H1 options provided per page
- [ ] Trust line or credibility signal in hero (where applicable)

## Page Copy
- [ ] Every wireframe section has complete copy
- [ ] Section headlines (H2) are clear and benefit-oriented
- [ ] Body copy matches the section purpose from wireframe
- [ ] No sections left blank or with placeholder text

## Social Proof
- [ ] Testimonials include name and role/company
- [ ] Results are specific (numbers, percentages, timeframes)
- [ ] No vague praise ("great service" without specifics)

## SEO Metadata
- [ ] Title tags on all pages (within 60 characters)
- [ ] Meta descriptions on all pages (within 160 characters)
- [ ] OG tags (title + description) on all pages
- [ ] Focus keyword identified per page

## Final Validation
- [ ] Complete copy package assembled (all pages, all sections)
- [ ] Copy reads well out loud (natural, not robotic)
- [ ] No spelling or grammar errors
- [ ] Client has reviewed the copy
- [ ] Client has approved (gate)
