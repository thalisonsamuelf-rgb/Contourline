# Pre-Launch Checklist

**Checklist ID:** `pre-launch-checklist`
**Phase:** QA / Launch
**Agent:** wp-developer
**Gate:** All items must pass before go-live

## Content
- [ ] All pages have final approved content (no placeholder text)
- [ ] All images are final and optimized
- [ ] All images have descriptive alt text
- [ ] Contact information is correct (phone, email, address)
- [ ] Copyright year is current
- [ ] Privacy policy page exists and is linked
- [ ] Terms of service page exists (if applicable)

## Functionality
- [ ] All contact forms submit correctly
- [ ] Form notification emails are received
- [ ] All internal links work (no 404s)
- [ ] All external links work and open in new tab
- [ ] Search function works (if applicable)
- [ ] Navigation works correctly on all devices
- [ ] 404 page displays custom design
- [ ] Favicon is set

## SEO
- [ ] Title tags on all pages (within 60 chars)
- [ ] Meta descriptions on all pages (within 160 chars)
- [ ] One H1 per page
- [ ] XML sitemap generated
- [ ] Robots.txt allows indexing (not staging robots.txt)
- [ ] Schema markup present
- [ ] Canonical URLs set
- [ ] Google Search Console verified

## Performance
- [ ] Lighthouse Performance >= 90
- [ ] Lighthouse Accessibility >= 90
- [ ] Lighthouse Best Practices >= 90
- [ ] Lighthouse SEO >= 90
- [ ] Core Web Vitals within targets
- [ ] Images optimized (WebP, compressed)
- [ ] Caching enabled and verified
- [ ] Lazy loading active

## Security
- [ ] SSL certificate ready for production
- [ ] Wordfence firewall active
- [ ] File editing disabled
- [ ] No "admin" username
- [ ] Login attempts limited
- [ ] Backup configured and tested
- [ ] WordPress and plugins up to date

## Analytics & Tracking
- [ ] Google Analytics (GA4) installed
- [ ] Google Search Console connected
- [ ] Analytics tracking verified (real-time test)
- [ ] Goal/conversion tracking configured (if applicable)

## Cross-Browser
- [ ] Chrome: PASS
- [ ] Firefox: PASS
- [ ] Safari: PASS
- [ ] Edge: PASS

## Mobile
- [ ] iPhone SE (375px): PASS
- [ ] iPhone 14 (390px): PASS
- [ ] Samsung Galaxy (412px): PASS
- [ ] iPad (768px): PASS

## Final Go/No-Go
- [ ] All above items pass
- [ ] QA report generated
- [ ] Client has approved staging
- [ ] Backup created before migration
- [ ] Rollback plan documented
- [ ] **VERDICT: GO / NO-GO**
