# Architecture Review Checklist

**Checklist ID:** `architecture-checklist`
**Phase:** Architecture
**Agent:** wp-architect
**Gate:** Client approval required before proceeding to Copywriting

## Sitemap
- [ ] All pages from discovery brief are represented
- [ ] Page hierarchy is logical (max 2 levels deep)
- [ ] URL structure is clean and SEO-friendly
- [ ] No orphan pages (every page reachable from navigation)

## Navigation
- [ ] Main navigation has 5-7 items maximum
- [ ] Navigation labels are clear and descriptive
- [ ] Footer navigation defined
- [ ] Mobile navigation behavior specified

## Page Frameworks
- [ ] Every page has an assigned framework (LP/institutional/lead-capture)
- [ ] Framework matches the page objective
- [ ] Framework rationale documented

## Wireframe Skeletons
- [ ] Wireframe exists for every page
- [ ] Each section has a clear purpose
- [ ] Section order follows the framework pattern
- [ ] Mobile stack order explicitly defined
- [ ] Notes for copywriter included (what copy goes where)
- [ ] Notes for designer included (layout hints)

## Consistency
- [ ] Page objectives are clear and non-overlapping
- [ ] No duplicate content across pages
- [ ] CTA destinations make sense (no dead ends)

## Final Validation
- [ ] Sitemap document complete
- [ ] Wireframe skeletons complete for all pages
- [ ] Client has reviewed the architecture
- [ ] Client has approved (gate)
