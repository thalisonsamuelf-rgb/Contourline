## Overview

Write hero headlines and subheadlines for every page following the Asimov Academy
copy style: action verb + clear benefit, bold and direct.

## Asimov Academy Copy Rules for Heroes

- H1: Start with action verb, max 8 words
- Subheadline: Supporting context, 12-15 words
- CTA: Specific action (never "Learn More")
- Trust line: One credibility signal (optional)
- Provide 2-3 alternative H1 options per page

## Action Items (per page)

### Step 1: Identify Page Objective
- [ ] What is the single goal of this page?
- [ ] What action should the visitor take?

### Step 2: Write H1 Headline
- [ ] Action verb + clear benefit
- [ ] Maximum 8 words
- [ ] Write 3 options, recommend the best

### Step 3: Write Subheadline
- [ ] Supporting context, 12-15 words
- [ ] Expands on the H1 without repeating

### Step 4: Define CTA
- [ ] Specific button text (verb + outcome)
- [ ] Define CTA destination (page/form/phone)

## Acceptance Criteria

- [ ] **AC-1:** Every page has a hero headline
- [ ] **AC-2:** All H1s start with an action verb
- [ ] **AC-3:** All H1s are 8 words or fewer
- [ ] **AC-4:** All CTAs are specific (no generic "Learn More")
- [ ] **AC-5:** 2-3 alternative options provided per page
