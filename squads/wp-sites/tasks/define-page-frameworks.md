## Overview

Assign one of three page frameworks to each page based on its objective.

## Page Frameworks

| Framework | Structure | Best For |
|-----------|-----------|----------|
| LP Product/Service | Hero -> Problem -> Solution -> Social Proof -> CTA | Conversion pages |
| Institutional | Flexible content sections | About, Services, Cases |
| Lead Capture | Hook -> Benefits -> Short Form | Landing pages with forms |

## Action Items

### Step 1: Evaluate Each Page
- [ ] Review page objective from sitemap
- [ ] Match objective to framework type
- [ ] Document rationale for assignment

### Step 2: Create Assignment Table
- [ ] Page name, framework, rationale for each page
- [ ] Ensure no page is unassigned

## Acceptance Criteria

- [ ] **AC-1:** Every page has an assigned framework
- [ ] **AC-2:** Framework matches the page objective
- [ ] **AC-3:** Assignment table is documented
