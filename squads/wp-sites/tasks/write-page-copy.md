## Overview

Write complete copy for every section of every page, following the wireframe
skeleton section order. Each section gets: headline, body copy, CTA.

## Asimov Academy Copy Rules

- Short paragraphs (2-3 sentences max)
- Benefits over features
- Problem-solution framing
- Second person (you/your)
- Specific numbers in social proof
- One CTA per section

## Action Items (per page, per section)

### For each section in wireframe skeleton:
- [ ] Write section headline (H2)
- [ ] Write body copy (paragraphs or bullets)
- [ ] Write CTA text and button label
- [ ] Add designer notes (emphasis, layout hints)

### Section-specific guidelines:
- **Problem section:** Name the pain point directly
- **Benefits section:** 3-4 benefit blocks with icon-friendly titles
- **Social proof section:** Named testimonials with specific results
- **CTA section:** Urgency without pressure, clear next step

## Acceptance Criteria

- [ ] **AC-1:** Every wireframe section has complete copy
- [ ] **AC-2:** Every section has a CTA
- [ ] **AC-3:** Benefits use problem-solution framing
- [ ] **AC-4:** Social proof includes specific numbers or names
- [ ] **AC-5:** Paragraphs are 2-3 sentences maximum
- [ ] **AC-6:** Copy is in second person (you/your)
