# Write SEO Metadata

**Task ID:** `write-seo-metadata`
**Pattern:** HO-TP-001
**Version:** 2.0
**Last Updated:** 2026-03-25

## Task Anatomy

| Field | Value |
|-------|-------|
| **task_name** | Write SEO Metadata |
| **status** | `pending` |
| **responsible_executor** | wp-copywriter |
| **execution_type** | `Agent` |
| **input** | Page copy package, target keywords |
| **output** | SEO metadata for every page |
| **action_items** | 5 items per page |
| **acceptance_criteria** | 4 criteria |

## Action Items (per page)

- [ ] **Title Tag:** Max 60 characters, include primary keyword
- [ ] **Meta Description:** Max 160 characters, include CTA and keyword
- [ ] **OG Title:** Social sharing headline
- [ ] **OG Description:** Social sharing description
- [ ] **Focus Keyword + Secondary Keywords:** For Yoast/RankMath configuration

## Acceptance Criteria

- [ ] **AC-1:** Every page has title tag within 60 characters
- [ ] **AC-2:** Every page has meta description within 160 characters
- [ ] **AC-3:** OG tags exist for all pages
- [ ] **AC-4:** Focus keyword identified per page

## Handoff

| Attribute | Value |
|-----------|-------|
| **Next Task** | `define-visual-direction` |
| **Trigger** | Client approves complete copy package |
| **Executor** | wp-designer |
