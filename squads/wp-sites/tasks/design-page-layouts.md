## Action Items (per page, per section)

### For each section:
- [ ] Define layout at desktop (1200px)
- [ ] Define layout at tablet (768px)
- [ ] Define layout at mobile (375px)
- [ ] Specify typography sizes, spacing, colors
- [ ] Note focal point and CTA placement

## Acceptance Criteria

- [ ] **AC-1:** Every page section has layout specs at all 3 breakpoints
- [ ] **AC-2:** Specs use design tokens from visual direction
- [ ] **AC-3:** CTA placement is clear in every section
- [ ] **AC-4:** Specs are implementable in Elementor/Gutenberg
