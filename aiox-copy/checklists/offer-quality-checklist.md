# Offer Stack Quality Checklist

## Overview

Quality gate for offer stacks generated by the Copy Squad. Validates the complete offer structure: core product, bonuses, pricing, guarantee, and urgency/scarcity elements.

---

## Critical Checks (must pass ALL)

### Value Stack
- [ ] Core offer is clearly defined with specific deliverables?
- [ ] Each component has its own perceived value (dollar amount)?
- [ ] Total perceived value is at minimum 10x the asking price?
- [ ] Stack is presented in escalating value order?
- [ ] Each item answers "what does the buyer GET from this?"?

### Price Anchoring
- [ ] Higher price anchor is established before reveal?
- [ ] Anchor is believable (not absurdly inflated)?
- [ ] Comparison to alternatives or cost-of-inaction is present?
- [ ] Final price feels like a deal relative to the anchor?

### Guarantee
- [ ] Guarantee is present and prominently displayed?
- [ ] Terms are specific and unambiguous (duration, conditions)?
- [ ] Guarantee removes the buyer's primary risk objection?
- [ ] Guarantee language is confident (not defensive or legalistic)?

### Bonuses
- [ ] Each bonus solves a real adjacent problem?
- [ ] Bonuses are NOT filler (each has standalone value)?
- [ ] At least one bonus addresses a key objection?
- [ ] Bonuses have their own value anchoring?
- [ ] Bonus names are benefit-driven (not generic titles)?

### Urgency and Scarcity
- [ ] At least one urgency or scarcity element is present?
- [ ] Urgency is genuine and defensible?
- [ ] Deadline, quantity limit, or bonus expiration is specific?
- [ ] Consequence of waiting is explicitly stated?

### Risk Reversal
- [ ] Buyer risk is lower than seller risk?
- [ ] "Worse than money back" guarantee if possible (keep bonuses, etc.)?
- [ ] Objection "what if it doesn't work for me?" is answered?

## Red Flags (auto-fail if present)

- No guarantee of any kind
- Fake scarcity (countdown that resets, false "limited" claims)
- Unclear pricing or hidden fees
- No bonuses (bare offer with nothing extra)
- Value stack math does not add up or feels dishonest
- Guarantee buried in fine print or footnotes
- Offer requires reading legal terms to understand what you get

## Offer Type Variations

### Low-Ticket ($7-$97)
- [ ] Impulse-buy friendly (fast decision)?
- [ ] Guarantee is simple (30-day, no questions)?
- [ ] Bonuses don't overcomplicate a simple purchase?

### Mid-Ticket ($97-$997)
- [ ] Comparison to alternatives or professional services?
- [ ] Payment plan option mentioned?
- [ ] Guarantee duration matches consumption time?

### High-Ticket ($1,000+)
- [ ] Application or call-based process?
- [ ] ROI projection or case study included?
- [ ] Extended guarantee or performance guarantee?
- [ ] White-glove or support element?
