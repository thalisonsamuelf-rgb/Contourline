# Break-the-Glass Tactics

## Source
Jeremy Haynes - Killer Follow-Up System

## When to Use
- Calendário dos vendedores está vazio
- Revenue está abaixo da meta
- Período tipicamente lento
- Precisa de cash injection rápido

## When NOT to Use
- Já tem muito momentum
- Sales team sobrecarregado
- Revenue em alta

## Tactic 1: "Sent from iPhone" Email

### History
Esta tática gerou **40,000+ respostas em 48 horas** quando usada no Grant Cardone's office. Literalmente parou a empresa por 2 dias.

### Template

```
Subject: following up

Body:

are you still interested in [product/topic]?

- [first name]

Sent from my iPhone
```

### Key Elements
- Subject: **tudo minúsculo**, casual
- Body: **uma linha só**, tudo minúsculo
- Assinatura: **só primeiro nome** com dash
- Footer: **"Sent from my iPhone"** (copiar exatamente como aparece no iPhone)
- **Empurrar** unsubscribe/address para BEM embaixo (dar vários enters)

### Portuguese Version

```
Assunto: acompanhando

Corpo:

você ainda está interessado em [produto/tópico]?

- [primeiro nome]

Enviado do meu iPhone
```

### Why It Works
- Parece email pessoal, não marketing
- Curiosidade sobre quem está mandando
- Simplicidade desarma defesas
- Parece urgente/importante
- Não parece automação

### Expected Responses
- "Sim, ainda estou interessado!" → LEAD REATIVADO
- "Não, obrigado" → Pelo menos você sabe
- "Quem é você?" → Oportunidade de reconectar

### Warning
⚠️ **CUIDADO**: Isso pode criar um TSUNAMI de respostas.
- Tenha equipe pronta para processar
- NÃO use se já está sobrecarregado
- Prepare respostas padrão

## Tactic 2: "Where Did We Lose You?"

### Psychology
A maioria das respostas NÃO é feedback real. A maioria diz: **"Você não me perdeu!"** e então voltam para o processo de compra.

### Template - English

```
Subject: quick question

Body:

Hey [nome],

Wondering if I could get a quick one-sentence feedback
from you on where we lost you.

Would really appreciate it.

- [seu nome]
```

### Template - Portuguese

```
Assunto: pergunta rápida

Corpo:

Oi [nome],

Queria pedir um feedback rápido:
onde foi que perdemos você?

Uma frase só já ajuda.

Obrigado,
[seu nome]
```

### Alternative Version

```
Assunto: uma pergunta

Corpo:

[nome],

Percebi que não tivemos mais contato.

Me ajuda a entender: o que aconteceu?

- [seu nome]
```

### Expected Response Types

**Response Type 1: Feedback Real**
> "O preço estava acima do meu orçamento"

**Action:** Oferecer alternativa, pagamento, ou drop sell

---

**Response Type 2: Reativação**
> "Não me perderam! Estava viajando, mas ainda quero"

**Action:** Reagendar imediatamente

---

**Response Type 3: Fechamento**
> "Já comprei outra solução"

**Action:** Remover da lista, talvez pedir feedback sobre o concorrente

## Tactic 3: "For the Dogs" Sequence

### Philosophy
Todo mundo assume que ninguém trabalha em períodos mortos. MAS existe uma minoria de pessoas altamente motivadas trabalhando enquanto outros descansam. Fale DIRETAMENTE com elas.

### Best Times to Use
- Black Friday weekend
- Entre Natal e Ano Novo
- Feriados prolongados
- Carnaval (no Brasil)
- Qualquer período "morto"

### Template - English

```
Subject: for the hungry ones

Body:

[Nome],

I know most people are resting right now.

But you're not most people, are you?

If you're one of the few people working
while others are taking time off...

If you're using this time to prepare
for next year...

Then I have an opportunity for you.

[OFFER with short deadline]

Only for those who act between [date] and [date].

For everyone else: enjoy your holidays.
For you: let's talk.

[CTA]

- [Nome]
```

### Template - Portuguese

```
Assunto: para os famintos

Corpo:

[Nome],

Eu sei que a maioria das pessoas está descansando agora.

Mas você não é a maioria, né?

Se você é uma das poucas pessoas que está trabalhando
enquanto outros tiram folga...

Se você está usando esse tempo para se preparar
para o próximo ano...

Então tenho uma oportunidade para você.

[OFERTA com deadline curto]

Só para quem agir entre [data] e [data].

Para os outros: aproveitem as férias.
Para você: vamos conversar.

[CTA]

- [Nome]
```

### Variations

**With Incentive:**
```
Para quem fechar até [data]:
- Bônus X (valor Y)
- Desconto de Z%
- Acesso antecipado a [nova feature]
```

**For Black Friday:**
```
Assunto: enquanto todos estão comprando TV

Corpo:

[Nome],

Todo mundo está comprando TV e geladeira.

Você está construindo algo maior.

Por isso, entre hoje e segunda:
[OFERTA ESPECIAL]

Só para quem não para em feriado.

[CTA]
```

## Tactic 4: Preferred Channel Discovery

### When to Use
Quando NADA está funcionando. O lead não responde email, não atende telefone, some.

### The Approach
Blast em **TODOS os canais** simultaneamente com UMA mensagem simples:
- Email
- SMS
- DM Instagram
- DM LinkedIn
- DM Facebook
- WhatsApp (se tiver)

### Template

```
Oi [Nome],

Tentei contato algumas vezes sem sucesso.

Qual o melhor canal para falar com você?

- [Seu nome] da [Empresa]
```

### Expected Responses
- "Me manda no WhatsApp: [número]" → Agora você sabe!
- "Prefiro email: [email]" → Ajuste sua comunicação
- "Para de me mandar mensagem" → Pelo menos você sabe parar

### Why This Works
Todo mundo tem um canal preferido. Você pode estar mandando email, mas ele só vê DM. Este blast revela onde ele realmente presta atenção.

### Important Notes
- Pode parecer "agressivo" mas:
  - Se não querem, vão dizer
  - Se querem mas não viram, você descobre onde veem
  - Melhor do que perder o lead completamente

## Tactic 5: "Is This Dead?" Direct

### Template

```
Assunto: isso morreu?

Corpo:

[Nome],

Tentando entender se ainda faz sentido continuar a conversa
ou se devo seguir em frente.

Me avisa?

- [Nome]
```

### Alternative

```
Assunto: última tentativa

Corpo:

[Nome],

Última vez que vou te mandar mensagem sobre isso.

Se tiver interesse, responde esse email.

Se não, tudo bem - vou parar de te incomodar.

- [Nome]
```

### Psychology
A "última tentativa" cria urgência real. Muitas pessoas que estavam procrastinando respondem porque não querem "perder a oportunidade".

## When to Deploy Each Tactic

| Situação | Tática Recomendada |
|----------|-------------------|
| Lista grande, precisa de volume | Sent from iPhone |
| Leads que estavam quentes | Where Did We Lose You |
| Período de férias/feriado | For the Dogs |
| Lead específico sumiu | Preferred Channel |
| Último recurso | Is This Dead |
