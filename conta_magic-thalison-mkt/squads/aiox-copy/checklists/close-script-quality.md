# Close Script Quality Checklist

## Overview

Quality gate for close scripts generated by the Copy Squad. A close script is the specific section of a sales conversation focused on transitioning from presentation to commitment. Validates transition naturalness, objection preemption, closing technique, silence usage, and next steps clarity.

---

## Critical Checks (must pass ALL)

### Transition Naturalness
- [ ] Shift from presentation to close is seamless (no jarring pivot)?
- [ ] Transition uses a bridge phrase that feels conversational?
- [ ] Prospect's own words are echoed back as the bridge ("Based on what you told me about...")?
- [ ] Close feels like a natural next step, not a surprise attack?
- [ ] Energy and tone remain consistent through the transition?

### Objection Preemption
- [ ] Top 3 objections are addressed BEFORE the close (not after)?
- [ ] Price objection is preempted with value reframing earlier in the conversation?
- [ ] "I need to think about it" is neutralized with decision framework?
- [ ] "I need to talk to [spouse/partner/boss]" has a proactive handling path?
- [ ] Preemption feels educational, not defensive?

### Assumptive Close
- [ ] Language assumes the sale ("When you start..." not "If you decide...")?
- [ ] Questions move toward implementation details ("Which package works best?")?
- [ ] Confidence is projected without being pushy?
- [ ] Close provides 2-3 options (not a yes/no binary)?
- [ ] Each option leads to a purchase (choice of yeses)?

### Silence Usage
- [ ] Script includes explicit [PAUSE] markers after key questions?
- [ ] Silence is used after stating the price (let prospect respond first)?
- [ ] No rushing to fill silence with discounts or over-explaining?
- [ ] At least 3 moments where silence is the scripted response?

### Next Steps Clarity
- [ ] Immediate next step is 100% clear after a "yes"?
- [ ] Payment process is explained simply?
- [ ] Onboarding or delivery timeline is stated?
- [ ] Prospect knows exactly what happens in the next 24-48 hours?
- [ ] Follow-up path for "not yet" (not "no") is defined?

## Red Flags (auto-fail if present)

- Close comes as a surprise with no transition
- Only one closing attempt (if "no", conversation dies)
- Script instructs the seller to discount at first pushback
- No silence markers (seller fills every gap)
- Manipulative pressure tactics (false scarcity in a 1:1 setting, guilt trips)
- Next steps are vague ("we'll be in touch")
- No path for the "not now" prospect (burned bridge)

## Closing Technique Reference

| Technique | When to Use | Script Pattern |
|-----------|-------------|----------------|
| **Assumptive** | High-conviction prospect | "Which start date works better for you?" |
| **Alternative** | Prospect needs to feel in control | "Option A or Option B — which fits your goals?" |
| **Takeaway** | Prospect is on the fence | "This might not be the right fit for everyone..." |
| **Summary** | Complex offer, many benefits | "So you're getting X, Y, Z — shall we get started?" |
| **Urgency** | Genuine deadline exists | "The pricing changes on Friday. Want to lock this in?" |
| **Puppy Dog** | Prospect is risk-averse | "Try it for 30 days. If it's not right, no hard feelings." |

## Close Script Technical Checks

- [ ] Script includes vocal tonality notes (confident, warm, slow down)?
- [ ] Branching paths for "yes", "maybe", and "not now" are all defined?
- [ ] Rebuttal loops are limited (max 3 attempts before graceful exit)?
- [ ] Script can be delivered in under 5 minutes (close section only)?
- [ ] Follow-up cadence is specified for non-closed prospects?
- [ ] "Win-back" language is available for prospects who go silent?

Total >= 35/50 for approval (70% threshold)
