## 1. FOUNDATION (Antes de Construir)

### Avatar & Positioning
- [ ] Avatar definido com detalhes (nome, idade, profissão, dores, desejos)
- [ ] Nível de consciência identificado (Schwartz)
- [ ] Estágio de sofisticação do mercado avaliado
- [ ] Hook principal definido e testado
- [ ] Story do autor documentada (Epiphany Bridge)
- [ ] Offer estruturada com value stack

### Value Ladder
- [ ] Value Ladder completa mapeada
- [ ] Posição do livro na ladder definida (front-end)
- [ ] Próximo passo após o livro definido
- [ ] Back-end (high-ticket) estruturado
- [ ] Preços de cada nível definidos

### Funnel Hack
- [ ] 10+ funnels do nicho analisados
- [ ] Padrões de headlines identificados
- [ ] Padrões de order bumps identificados
- [ ] Padrões de upsells identificados
- [ ] Padrões de emails identificados

---

## 2. SALES PAGE (Free + Shipping)

### Header Section
- [ ] Headline com promessa clara e específica
- [ ] Subheadline que qualifica o avatar
- [ ] Imagem do livro (3D mockup profissional)
- [ ] Badge "GRÁTIS" visível
- [ ] Preço do frete claro

### Story Section
- [ ] VSL ou carta de vendas presente
- [ ] Hook que prende atenção em 3 segundos
- [ ] Problema/dor do avatar articulado
- [ ] Sua história de transformação (Epiphany Bridge)
- [ ] Credenciais e resultados mencionados
- [ ] Transição natural para a oferta

### Value Stack Section
- [ ] Valor de cada componente listado
- [ ] Bônus digitais incluídos
- [ ] Valor total calculado (âncora alta)
- [ ] Preço final (frete) muito menor que valor

### Social Proof Section
- [ ] Mínimo 3 testimonials de leitores
- [ ] Testimonials com nome e foto (se possível)
- [ ] Resultados específicos mencionados
- [ ] Números sociais (cópias vendidas, países, etc.)
- [ ] Logos de autoridade (se aplicável)

### Offer Section
- [ ] Resumo claro do que está incluído
- [ ] Garantia explícita (30 dias)
- [ ] CTA claro: "QUERO MEU LIVRO GRÁTIS"
- [ ] Escassez real (se aplicável)
- [ ] Urgência (se aplicável)

### Technical
- [ ] Mobile responsive
- [ ] Carregamento rápido (<3 segundos)
- [ ] Imagens otimizadas
- [ ] Formulário funcional
- [ ] SSL ativo

---

## 3. ORDER BUMPS

### Order Bump #1 (Audiobook ou similar)
- [ ] Checkbox presente no checkout
- [ ] Copy persuasivo (benefício de consumir diferente)
- [ ] Preço com desconto visível
- [ ] Valor âncora mostrado
- [ ] Bônus extra incluído

### Order Bump #2 (Workshop ou similar)
- [ ] Checkbox presente no checkout
- [ ] Copy focado em implementação mais rápida
- [ ] Preço com desconto significativo
- [ ] Lista de o que está incluído
- [ ] Valor âncora mostrado

### Métricas Target
- [ ] Order bump #1: Target 20-30%
- [ ] Order bump #2: Target 10-20%
- [ ] Combinado: Target 30-50%

---

## 4. UPSELL SEQUENCE

### Upsell #1 (OTO)
- [ ] Página de upsell criada
- [ ] Vídeo curto (3-5 min) gravado
- [ ] Oferta complementar ao livro
- [ ] Preço com desconto significativo
- [ ] Urgência: "Só aparece uma vez"
- [ ] Botão de aceitar claro
- [ ] Botão de recusar presente
- [ ] Redireciona corretamente

### Downsell #1
- [ ] Página de downsell criada
- [ ] Versão mais barata/parcelada
- [ ] Copy empático ("Entendo...")
- [ ] Opção de recusar para thank you

### Upsell #2 (Opcional)
- [ ] Página criada (se aplicável)
- [ ] Oferta de maior valor
- [ ] Vídeo mais longo (5-7 min)
- [ ] Stack de valor completo
- [ ] Garantia incluída

### Métricas Target
- [ ] OTO #1: Target 10-20%
- [ ] Downsell #1: Target 5-10%
- [ ] OTO #2: Target 5-10%
