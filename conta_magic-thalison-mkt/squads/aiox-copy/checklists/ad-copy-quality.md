# Ad Copy Quality Checklist

## Overview

Quality gate for ad copy generated by the Copy Squad. Covers paid social, search, display, and native ad formats. Validates hook, platform fit, CTA clarity, targeting alignment, and compliance.

---

## Critical Checks (must pass ALL)

### Hook (First 3 Seconds)
- [ ] First line stops the scroll in a feed environment?
- [ ] Hook is visual or verbal (matches ad format)?
- [ ] Pattern interrupt is genuine (not clickbait)?
- [ ] Viewer knows the topic/benefit immediately?
- [ ] Hook connects to the target audience's current state?

### Platform Fit
- [ ] Copy matches the platform's native tone and format?
- [ ] Character limits are respected for the platform?
- [ ] Format follows platform best practices (carousel, single image, video)?
- [ ] Copy feels organic in the feed (not "ad-like")?
- [ ] Hashtags, emojis, or formatting match platform norms?

### CTA Clarity
- [ ] One unmistakable action the viewer should take?
- [ ] CTA language is specific ("Get the free guide" not "Learn more")?
- [ ] CTA matches the ad destination (no bait-and-switch)?
- [ ] Button/link text aligns with the ad copy promise?

### Targeting Alignment
- [ ] Copy speaks directly to the ad's target audience segment?
- [ ] Awareness level matches the audience (cold vs warm)?
- [ ] Language resonates with the demographic (age, profession, culture)?
- [ ] Pain points and desires match what the audience cares about?
- [ ] Ad would not resonate with everyone (properly targeted)?

### Compliance
- [ ] No prohibited claims (health, income, guarantees per platform)?
- [ ] Disclaimers present where required?
- [ ] Landing page consistency with ad claims?
- [ ] No discriminatory targeting language?
- [ ] Political/social content guidelines followed?

## Red Flags (auto-fail if present)

- Hook could be for any product (generic attention grab)
- Copy exceeds platform character limits
- Multiple CTAs competing for attention
- Bait-and-switch between ad and landing page
- Claims that violate platform advertising policies
- Copy tone feels completely wrong for the platform
- No connection between the hook and the offer

## Platform-Specific Checks

### Facebook/Instagram
- [ ] Primary text under 125 characters for full display (or strong first line)?
- [ ] Headline under 40 characters?
- [ ] Description adds value (not redundant)?
- [ ] Less than 20% text on image (if applicable)?
- [ ] Engagement-friendly (would get likes/comments)?

### Google Search
- [ ] Headlines use keywords naturally?
- [ ] Description includes the core benefit and CTA?
- [ ] Extensions (sitelinks, callouts) are utilized?
- [ ] Ad matches search intent?

### YouTube
- [ ] First 5 seconds are unskippable (compelling hook)?
- [ ] Script works with and without sound?
- [ ] CTA appears both verbally and visually?
- [ ] Companion banner aligns with video message?

### LinkedIn
- [ ] Professional tone without being boring?
- [ ] Value proposition is B2B-relevant?
- [ ] Social proof is industry-specific?
- [ ] CTA is appropriate for professional context?

---

## Ad Variation Checks
- [ ] At least 3 hook variations are provided for testing?
- [ ] Variations test different angles (pain, desire, curiosity, social proof)?
- [ ] Each variation is genuinely different (not word swaps)?

---

## Minimum Score

Total >= 35/50 for approval (70% threshold)
