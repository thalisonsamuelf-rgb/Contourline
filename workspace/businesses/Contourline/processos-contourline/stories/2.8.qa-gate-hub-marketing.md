# Story 2.8: QA Gate — Validação e Testes do Hub de Marketing Completo

## Status
Draft

executor: "@qa"

quality_gate_tools: [test report, regression tests, user acceptance testing]

## Story
**As a** Head de Growth Marketing (Thalison) e time da Contourline,
**I want** uma validação completa do hub de marketing antes de colocá-lo em uso pela equipe inteira,
**so that** o sistema funcione de forma confiável, segura e que o time realmente consiga usar no dia a dia.

## Acceptance Criteria

1. Todos os fluxos das Stories 2.2 a 2.6 testados de ponta a ponta sem falhas críticas
2. Integração entre hub → pipeline de vídeo → CRM → social media → dashboard funcionando corretamente
3. Performance: hub responde em menos de 3 segundos para as operações principais
4. Segurança: autenticação e autorização por papel validados (criador / revisor / supervisor)
5. Teste de aceitação com usuários reais: pelo menos 3 membros do time usaram o sistema por 1 semana
6. Dados do dashboard batem com os dados das plataformas de origem (tolerância de 1%)
7. Playbook (Story 2.7) validado e acessível para o time antes do go-live
8. Plano de rollback documentado caso algum sistema crítico falhe em produção
