---

## Descrição

Esta story é o QA Gate de encerramento do Epic 10 — Novos Negócios & Internacional. É o último epic do roadmap da Contourline (P4), executado após todos os Epics P1-P3 estarem em andamento. Este QA Gate tem natureza diferente dos demais: valida não apenas a qualidade técnica dos sistemas, mas também a prontidão operacional da Contourline para operar internacionalmente com suporte de dados.

O gate valida: (1) o mapeamento da operação internacional está completo e confiável, (2) o pipeline de novos negócios está ativo e sendo gerido, (3) o COMEX está sendo rastreado, (4) o dashboard internacional está disponível para o CEO e conselheiros, e (5) o primeiro ciclo de inteligência de mercado foi executado.

- [ ] Sistema de rastreamento COMEX com ao menos 2 importações reais sendo rastreadas

## Checklist de Validação

### 7 Quality Checks do QA Gate

**QC1 — Completude do Mapeamento (Alta)**
- [ ] Documento de operação internacional aprovado por Filipe Ferreira
- [ ] Framework de avaliação de mercados documentado e aplicável
- [ ] Processos de COMEX documentados com fluxogramas
- [ ] Top 3 mercados priorizados com justificativa

**QC2 — Pipeline de Novos Negócios (Alta)**
- [ ] Funil "Novos Negócios Internacional" ativo no RD Station
- [ ] Ao menos 3 oportunidades cadastradas com campos completos
- [ ] Notificações de mudança de etapa funcionando (testar com oportunidade de teste)
- [ ] Relatório semanal chegando para Filipe Ferreira
- [ ] Dashboard de pipeline no Metabase com dados reais

**QC3 — Automação COMEX (Alta)**
- [ ] Ao menos 2 importações reais sendo rastreadas no sistema
- [ ] Alertas de atraso testados e chegando para Julio Cesar (Analista de Importacao/Comercio Exterior)
- [ ] Checklist de documentação funcionando por tipo de importação
- [ ] Dashboard COMEX com dados reais

**QC4 — Dashboard Internacional (Alta)**
- [ ] Dashboard "Operações Internacionais" carregando em < 5 segundos
- [ ] 5 painéis com dados reais (validar spot check de 10 registros vs Supabase)
- [ ] Relatório mensal para conselheiros testado e sendo enviado
- [ ] CEO Egio consegue acessar o dashboard sem suporte de TI

**QC5 — Inteligência de Mercado (Média)**
- [ ] Workflow de pesquisa de mercado executado para ao menos 1 mercado
- [ ] Relatório de mercado gerado por IA com fontes citadas
- [ ] Relatório revisado e aprovado por Filipe Ferreira
- [ ] Processo trimestral documentado com próximas datas definidas

**QC6 — Segurança e Confidencialidade (Alta)**
- [ ] Dados de pipeline de novos negócios acessíveis apenas para time autorizado
- [ ] Dados de COMEX (custos) com acesso restrito ao financeiro e diretoria
- [ ] Relatórios de inteligência de mercado não são públicos
- [ ] Credenciais de APIs (LLM, EXA) não expostas em código ou logs

**QC7 — Operabilidade Internacional (Alta)**
- [ ] Filipe Ferreira consegue: adicionar nova oportunidade, atualizar etapa, iniciar pesquisa de mercado — sem suporte de TI
- [ ] Responsável COMEX consegue: registrar nova importação, atualizar etapa — sem suporte de TI
- [ ] Documentação técnica cobrindo todos os sistemas do epic
- [ ] Plano de contingência para cada automação (o que fazer se falhar)

## Tasks Técnicas

**T10.6.1 — Preparar sessão de validação com stakeholders**

- Agendar: sessao com Julio Cesar — Analista de Importacao/Comercio Exterior (1h) para validar story 10.3

**T10.6.2 — Executar QC1 — Completude do Mapeamento**
- Verificar que documento de operação internacional está publicado e aprovado
- Verificar que framework de scoring está aplicado aos 3 mercados top
- Verificar que processos COMEX têm fluxograma aprovado pelo responsável

**T10.6.3 — Executar QC2 — Pipeline**
- Verificar ao menos 3 oportunidades no funil RD Station
- Forçar mudança de etapa em oportunidade de teste → notificação chega?

**T10.6.4 — Executar QC3 — COMEX**
- Verificar 2 importações reais no Supabase com dados completos
- Simular atraso de etapa → alerta chega para responsável?
- Verificar dashboard COMEX com dados corretos

**T10.6.5 — Executar QC4 — Dashboard Internacional**
- Medir tempo de carregamento dos 5 painéis
- Spot check: 10 registros no dashboard vs queries Supabase diretas
- Verificar que CEO consegue acessar via login próprio
- Verificar relatório mensal chegou para conselheiros no último mês (ou testar manualmente)

**T10.6.6 — Executar QC5 — Inteligência de Mercado**
- Verificar relatório de 1 mercado gerado e aprovado no repositório
- Testar workflow de pesquisa para um segundo mercado
- Verificar que fontes estão citadas no relatório
- Confirmar que processo trimestral está calendariado

**T10.6.7 — Executar QC6 — Segurança**
- Testar acesso com usuário não autorizado a dados de pipeline e COMEX → negado?
- Verificar que relatórios de inteligência de mercado não são acessíveis publicamente
- Revisar N8N: credenciais hardcoded? (não deve ter)

**T10.6.8 — Executar QC7 — Operabilidade**
- Sessão ao vivo: Filipe tenta adicionar nova oportunidade e iniciar pesquisa de mercado sem ajuda
- Sessao ao vivo: Julio Cesar (Analista de Importacao/Comercio Exterior) tenta registrar nova importacao sem ajuda
- Verificar documentação técnica: completa para todos os sistemas do epic?

**T10.6.9 — Consolidar e emitir veredicto**
- Compilar resultados dos 7 QCs
- Classificar: PASS / CONCERNS / FAIL
- Emitir veredicto global com lista de issues e planos de ação
